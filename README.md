### Features of the project

## Admin Can     -------------------------------------------------------- 

1. Add student                               
2. Remove student 
3. Add lectures 
4. Remove lectures  
5. Add Assignments 
6. Remove Assignments 

## Student Can  --------------------------------------------------------

1. Can see lectures 
2. Filter lectures
3. Can see assignments
4. Filter assignments
